AI Experimentation

Technical Understanding

The goal for this assignment was to create a better design for my portfolio. I felt okay about my initial design, but I wanted to create something more visually interesting. I had a difficult time at first, as I uploaded an image as an example of what I wanted my portfolio to look like unto Claude, but it then gave me a zip file that would have erased the work I had already done. I decided instead to primarily work on the CSS and keep the bones of my original portfolio. I wanted my portfolio to include more of an editorial look, and I believe that I was able to achieve this. The first attempt at changing the design of my portfolio was unsuccessful because it was using made up names, such as .slide, and .page, and my HTML did not use these terms so my project looked worse than it originally did. My HTML page used different tags such as header or footer and so I had to readjust my code to fit these tags. I was surprised to see that the colors were blue text and an odd beige, as the instructions included pink. The reason for this was because the code was not able to link itself to the code in my index, and this was causing pieces of my portfolio to have default features. The second attempt was a lot closer to what I was hoping to see, but was not fully there. The colors and fonts started appearing, which gave me hope. I found out that my index was structured in a way where the navigation comes before my header, and therefore my code was having a hard time finding where to link. I was able to fix these issues once I shared a deeper look into my index file. The AI was able to look at the code and get a better idea of what was happening. Once this was done, I was able to rewrite the CSS page in a way that allowed it to handle the navigation bar and apply my design. I ultimately ended up only changing the CSS instead of the HTML file, but I will play with the HTML file again later on.

Process Evaluation

I was surprised at how fast AI was able to develop code for this project. As a student I appreciated the extra assistance, but as a student in the humanities it made me realize how very little knowledge on code is needed to clean a csv or design your portfolio. However, the AI I used was not able to create what I wanted without specifics, and I think design is still an aspect that AI may struggle with. Cleaning the .csv was smoother than getting a design I was satisfied with. It is fair to say that AI has the ability to create a good design, however, it needs assistance and clear instructions. AI allows the creation of digital spaces to be an easier process, but I did not enjoy when AI tried to give me a .zip file that would redo my project, instead I wanted it to walk me through the process. I learned about the structure of my code, but only by asking questions, and engaging with the code being provided.
